ce — context engine

Generic ingest of mixed enterprise data into a queryable fact store, with markdown views, conflict resolution, and BM25/graph retrieval.

Browse

Entity browser
Walk types → ids → facts. Like a file tree over the corpus.
Conflict inbox
Side-by-side disagreeing facts with target labels & provenance.
Resolutions inbox
Approve / reject LLM-resolved entity references.

HTTP API

  • GET /entities — list entity types with counts
  • GET /entities/{type}?q=&offset=&limit= — paginated id+label list
  • GET /entities/{type}/{id} — entity facts (with ref labels) + rendered markdown
  • POST /retrieve — BM25 search + optional graph hops
  • GET /conflicts · GET /conflicts/{id} · POST /conflicts/{id}/resolve
  • GET /resolutions · POST /resolutions/approve · POST /resolutions/reject
  • GET /health
